NamuWiki Source Bookmarklet

아래 파란 링크를 북마크 바로 드래그하세요. 이후 나무위키 편집/원문 화면에서 북마크를 누르면 현재 편집창의 위키 원문을 .code 파일로 다운로드합니다.

Namu source to .code

드래그가 안 되면 아래 텍스트를 새 북마크의 URL에 붙여넣어도 됩니다.

사용 순서

  1. 이 페이지의 파란 링크를 북마크 바로 드래그합니다.
  2. 나무위키 문서에서 편집/원문 화면을 엽니다.
  3. 방금 만든 북마크를 클릭합니다.
  4. 다운로드된 .code 파일을 server/scripts/namu_pastes에 넣습니다.
  5. npm.cmd run import:namu:all을 실행합니다.
